# Email Campaigns

Emails rédigés pour les campagnes Systeme.io (base YouTube lead magnets).

## Structure
- Chaque email est un fichier markdown avec la date et le sujet
- Les placeholders `[SCREENSHOT: description]` indiquent où insérer des images
- Les liens `LIEN_SKOOL` sont à remplacer par le vrai lien

## Workflow
1. Review le markdown
2. Ajouter les screenshots aux emplacements indiqués
3. Copier-coller dans Systeme.io
